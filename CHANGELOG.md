# Changelog

Все важные изменения в проекте документируются в этом файле.

## [1.0.2] - 2024-11-25

### Исправлено
- ✅ Исправлена совместимость с Next.js 15
- ✅ Обновлен тип `PageProps` для асинхронных `params`
- ✅ Компонент `ArticlePage` теперь асинхронный
- ✅ Функция `generateMetadata` обновлена для работы с Promise params

### Технические детали
В Next.js 15 изменился API динамических маршрутов:
- **Было:** `params: { slug: string }`
- **Стало:** `params: Promise<{ slug: string }>`

Это требует использования `await params` в компонентах.
